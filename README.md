# 689 Cats iPhone PWA – Foundation 0.1

Diese Version ist für iPhone-only Workflow gedacht. Kein lokales Terminal nötig, wenn du sie über GitHub + Vercel veröffentlichst.

## Ablauf mit iPhone

1. ZIP herunterladen.
2. Auf GitHub ein neues Repository erstellen.
3. ZIP-Inhalt ins Repository hochladen.
4. Vercel mit GitHub verbinden.
5. In Vercel zwei Environment Variables setzen:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
6. Deploy starten.
7. URL auf dem iPhone in Safari öffnen.
8. Teilen → Zum Home-Bildschirm.

## Wichtig in Supabase

Das Schema V1.0 muss bereits im SQL Editor ausgeführt sein.
